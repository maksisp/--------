# Пятнашки


## Запуск проекта

Для запуска проекта вам потребуется Python. Убедитесь, что он установлен на вашем компьютере.

1. **Установите зависимости**:

    ```bash
    pip install -r requirements.txt
    ```

2. **Запустите main.py**:

    ```bash
    python main.py
    ```

Это запустит основной файл вашего проекта.
